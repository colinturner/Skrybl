# Scribbling

## An app for learning foreign languages through dictation.

### MVP user stories:

```
As a user,
So that I can hear the dictation,
I can press play on an audio file.

As a user,
So that I can write down my solution,
There is an input field for me to write in.

As a user,
So that I can check my work for accuracy,
I can press 'Submit' when I'm done.

As a user,
So that I can check my work for accuracy,
I can see the solution text when I hit 'Submit'.
```


### Nice-to-haves:

```
As a user,
So that I can easily verify my accuracy,
The words I got wrong are highlighted in the solution.

As a user,
So that I can expand my vocabulary,
I can see translations of foreign words by hovering over them.

As a user,
So that I can practice with varied material,
I can hit a button to generate a new audio file challenge.

As a user,
So that I can worry about vocabulary only,
I can disable punctuation so that it's not necessary in the solution.
```
